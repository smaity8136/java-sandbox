This module will give an example of how to use Spring's build in EmbeddedDatabaseBuilder class to simulate CRUD operations to an HSQL in-memory database.

The EmbeddedDatabaseBuilder class is part of the org.springframework.boot:spring-boot-starter-jdbc module. It allows us to create a java.sql.Datasource instance that is configured to point to a HSQLDB in-memory database. Convienient for testing.
